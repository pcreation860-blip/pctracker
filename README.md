# PCTracker - Production Tracking App

## 🚀 Quick Setup (5 minutes)

### Step 1 — Install Node.js
Download from: https://nodejs.org (click the LTS button)

### Step 2 — Open this folder
- Extract this ZIP to your Desktop
- Open the folder `pctracker`

### Step 3 — Open Terminal in this folder
- Windows: Right-click inside folder → "Open in Terminal"
  OR press Win+R → type `cmd` → `cd Desktop\pctracker`

### Step 4 — Run these commands
```
npm install
npm run dev
```

### Step 5 — Open your app
Go to: http://localhost:5173

---

## 🌍 Deploy to Internet (FREE - Vercel)

### Option A — Drag and Drop (Easiest)
1. Run `npm run build` in terminal
2. Go to https://vercel.com → Sign up free
3. Drag the `dist` folder to Vercel
4. Done! You get a free link like yourapp.vercel.app

### Option B — Via GitHub (Recommended for updates)
1. Create account at https://github.com
2. Create new repository called `pctracker`
3. Upload all these files to GitHub
4. Go to https://vercel.com → Import from GitHub
5. Click Deploy
